MySearch

Tutorial: JavaServer Faces library for MasterKey

This project contains samples files to accompany the written tutorial on how to build a simple Pazpar2 search interface using the library 'mkjsf'.

This file (mysearch-01.xhtml) is the first sample, merely containing the namespaces to be used throughout the tutorial. The following samples mysearch-02.xhtml through mysearch-07.xhtml can be found in the sub-directory tutorial-files.
